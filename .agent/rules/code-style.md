# Code Style for Food Mood

1.  **TypeScript**: Strict mode enabled. Use explicit types or interfaces for all props, states, and return values where inference is not obvious.
2.  **Tailwind CSS**: Use utility classes. Avoid custom CSS files unless absolutely necessary.
3.  **Components**: Use functional components with hooks.
4.  **Documentation**: Use JSDoc for complex functions and components.
5.  **Imports**: Use absolute imports (`@/components/...`, `@/lib/...`).
6.  **Language**: All user-facing text and i18n should be in Spanish.
